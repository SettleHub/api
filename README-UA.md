# esettlement WEB API

esettltment - це система електронного поселення в гуртожитки.

Основне завдання - спростити процес подачі документів на поселення в гуртожиток абітурієнтами/студентами.

Зберіть з вихідного коду за допомогою [інструкції по збірці](./BUILD.md)

Або використовуйте контейнер Docker, див. [інструкцію по docker](./DOCKER.md)
